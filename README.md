# Серверная часть проекта

## Установка

1. Перейдите в папку `server`:
   ```bash
   cd server
2. Установите зависимости
      ```bash
   npm install

## Запуск

1.	Запустите серверную часть в режиме разработки:
      ```bash
      npm run dev

2.	Серверная часть будет доступна по адресу:
      ```bash
      http://localhost:3000
